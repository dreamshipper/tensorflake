# Tensorflake Website

This is the official website for **Tensorflake**, hosted on [Cloudflare Pages](https://pages.cloudflare.com).  
It is a simple static site built with **HTML, CSS, and JavaScript**, featuring three pages:

- **Home** – Share articles and insights regularly.
- **About Me** – Introduce yourself and your background.
- **Contact** – Provide a contact form for visitors.

## 🚀 Deployment on Cloudflare Pages

1. Push this repository to GitHub.
2. Go to your **Cloudflare Dashboard → Pages → Create Project**.
3. Connect your GitHub repo.
4. Configure build settings:
   - **Build command:** none (static site)
   - **Output folder:** root
5. Deploy — your site will be live under **tensorflake.com**.
